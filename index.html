2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. You have to use consistent and clear indentation to make it more readable.
  2. You have to use meaningful names for variables and functions that describe their purpose. This should make your code self-documenting and easier to understand.
  3. You have to add comments to explain complex or non-obvious parts of your code. Well-placed comments should help other developers (and your future self) understand your code.
  4. You have to minimize the use of global variables to prevent naming conflicts and maintain data encapsulation. Use local or function-scoped variables whenever possible.
  5. You have to follow a consistent coding style, whether it's your own, your team's, or a recognized style guide like "JavaScript Standard Style" or "Airbnb JavaScript Style Guide." This ensures uniformity and readability across your codebase.

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // s přiřazením prázdné (žádné) hodnoty
let cele = 51; // celé číslo (typ number)
let realne = 3,9; // reálné číslo (typ number)
let desetinne = 6e7; // zápis desetinného čísla pomocí exponentu
let text = "Dobrý den"; // typ string
let prestupny = false; // proměnná typu boolean, můžou pouze nabýt hodnoty true nebo false

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, /=, *=, %=, ++, --, += (pro spojování řetězců)

Logické

&& (a zároveň), || (nebo), ! (negace), != (nerovná se), ? : (ternární operátor)

Porovnávací

== (rovná se), === (strikně se rovná, posuzuje i datový typ)
Pro porovnávání čísel:
< (menší), <= (menší nebo rovno), > (větší), >= (větší nebo rovno)
Větší a menší se také používá k porovnávání lexikografického pořadí

Bitové

&, |, ~, ^, <<, >>, >>>

Typové

typeof (typ proměnné), instanceof (), cosntructor

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

// Zobrazí se vyskakovací okno se zprávou Nezadal jsi správné heslo!
alert('Nezadal jsi správné heslo!');

prompt()

// Do proměnné password vloží heslo zadané uživatelem
let password = prompt('Zadej heslo:');

confirm()

// Zobrazí se vyskakovací okno, které se uživatele ptá, jestli chce přesměrovat na stránku seznam.cz
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:

  1. Prohlížeč může vyskakovací okna blokovat.
  2. Stránky s vyskakovacími okny nevypadá důvěryhodně.
